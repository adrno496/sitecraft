Questions fréquentes

FAQ

Qu'est-ce que je reçois exactement ?

Vous recevez un dossier ZIP contenant tous les fichiers de votre site : pages HTML, CSS, sitemap.xml, robots.txt, et deux guides PDF — l'un pour mettre en ligne, l'autre pour modifier. Vous possédez 100 % du code source.

Combien de temps pour recevoir mon site ?

72h ouvrées après validation du brief et du paiement. Un aller-retour de correction est inclus (deux avec l'offre Premium). Garanti.

Y a-t-il des frais cachés ou un abonnement ?

Non. Prix fixe, paiement unique. Le site t'appartient à 100 %. Aucun abonnement, aucun frais récurrent.

Est-ce que le site sera bien référencé sur Google ?

Oui. Chaque site inclut un SEO on-page complet : balises meta, structure Hn, vitesse optimisée, données structurées schema.org, sitemap.xml et robots.txt.

Puis-je modifier le site moi-même après livraison ?

Oui, le guide de modification PDF vous explique comment changer textes, images, couleurs. Le code est commenté et lisible. Ou contactez-nous pour une mise à jour payante si vous préférez déléguer.

Comment mettre le site en ligne ?

Le guide PDF accompagne étape par étape : GitHub (gratuit) + Vercel (gratuit). Votre site est en ligne en moins de 15 minutes. Connexion domaine incluse dans le guide. Option mise en ligne complète disponible à 100 €.

Proposez-vous l'hébergement ou la maintenance ?

L'hébergement est gratuit via GitHub + Vercel (expliqué dans le guide). Maintenance disponible à 29 €/mois pour les mises à jour de contenu et le suivi technique.

Puis-je ajouter des options après le questionnaire ?

Oui. Les options (maintenance, rédaction SEO, setup Google Business, livraison prioritaire) peuvent être ajoutées avant la validation finale. Mentionnez-les dans le questionnaire ou répondez à l'email de proposition.
